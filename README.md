# 見やすいWebページを作ろう — Gemma があなたのデザインを分析する体験型ゲーム

千葉工業大学 2026年前期「Web3概論」の第2回課題（テーマ：産官学プロダクトを作ろう）として作成した、体験型Webアプリケーションです。

## プロジェクト概要
本プロジェクトは、オープンキャンパス等で高校生や一般の方に「デザインの面白さ」と「AI（LLM）の凄さ」を同時に体験してもらうことを目的としたゲームです。プレイヤーが直感的にWebページをデザインし、その結果をGoogleのAI「Gemma」が認知科学的な視点から評価します。

## 課題提出内容
- **選択テーマ**: 産（楽しんでくれそうなゲーム）/ 官（学びが大きくなるツール）
- **一言紹介**: 千葉工業大学認知情報科学科のオープンキャンパスにおける学び体験コーナーで使えるような、Webページを動的にデザインするゲームを作成した。評価にはGoogle Gemini APIを用いている。わかりやすいUIと学科の学習内容に関連させることを心掛けた。
- **公開URL (Vercel)**: `https://web-design-evaluator-gemma.vercel.app/`

## 主な機能と特徴
1. **リアルタイム・デザインプレビュー**: 14種類のパラメータ（色、文字サイズ、余白など）をスライダーで操作し、瞬時に画面右側のプレビューに反映させます。
2. **Gemma による多角的なAI診断**: 視認性・構成・認知負荷の3つの観点から、AIがプロのコンサルタントとしてスコアと改善アドバイスを出力します。
3. **学習コンテンツ（解説画面）**: 「なぜこの余白が良いのか」といった認知科学的アプローチや、裏側で動いているプロンプトエンジニアリングの仕組みを解説するページを実装しています。

## 開発・動作環境
- **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript, Chart.js
- **Backend**: Python 3, FastAPI, Uvicorn, Google GenAI SDK (Gemini API / Gemma 3)
- **AI Assistant**: Antigravity (ペアプログラミングによる構築)
- **Deployment**: Vercel

## ファイル構成
主要なファイル構成は以下の通りです。

```text
web-geme/
├── main.py                  # FastAPI サーバー本体・AI連携バックエンド
├── grade.py                 # スコア判定ロジック
├── requirements.txt         # Python依存ライブラリ
├── prompt_body.txt等        # 評価用AIプロンプト（3視点分）
├── static/
│   ├── index.html           # SPAのエントリーポイント
│   ├── css/                 # 画面ごとのスタイルシート
│   ├── js/                  # フロントエンドロジック (app.js, evaluator.js等)
│   └── pages/               # 動的に読み込まれるHTMLコンポーネント
├── DOCUMENT.md              # 開発者向けの詳細な仕様・設計ドキュメント
└── README.md                # 本ファイル
```
※ 詳細な設計、パラメータのマッピング、AI評価のアーキテクチャについては、[`DOCUMENT.md`](./DOCUMENT.md) をご参照ください。
